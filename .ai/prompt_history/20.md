- **Branch/Tag/Release:** Branch oluşturma/listeleme, tag ve release yönetimi
- **Arama:** Kod, issue, PR, repo ve kullanıcı araması

**Örnek:** "Bu repo için yeni bir issue oluştur", "Açık PR'ları listele", "main branch'e dosya pushla"

---

## 2. Postman MCP (`mcp_postman_mcp_*`)

Postman workspace, collection, environment, mock server ve API spec yönetimi sağlar.

**Temel Kullanım Alanları:**
- **Collection:** Oluşturma, okuma, güncelleme, çoğaltma, çalıştırma, request ekleme/güncelleme
- **Environment:** Oluşturma, okuma, güncelleme
- **API Spec:** Spec oluşturma/okuma/güncelleme, spec dosyası yönetimi, collection ile senkronizasyon
- **Mock Server:** Mock oluşturma, güncelleme, yayınlama
- **Workspace:** Oluşturma, okuma, güncelleme
- **Otomatik üretim:** Collection'dan spec veya spec'ten collection üretme

**Örnek:** "Product service için Postman collection oluştur", "Collection'ı çalıştır", "Mock server oluştur"

---

## 3. Copilot Upgrade MCP (`mcp_copilot_upgra_*`)

.NET projelerinin upgrade/modernizasyon süreçlerini yönetir.

**Temel Kullanım Alanları:**
- Proje analizi, plan üretimi, SDK-style dönüşüm, upgrade çalıştırma
- Build, test, hata düzeltme, paket versiyon doğrulama
- NuGet feed authentication, state yönetimi

**Not:** Bu proje Java/Spring tabanlı olduğundan bu MCP seti doğrudan kullanılmaz.

---

## 4. GitHub Copilot AppMod MCP (`mcp_github_copilo_appmod-*`)

Uygulama modernizasyonu ve Azure'a taşıma süreçleri için kullanılır.

**Temel Kullanım Alanları:**

- Azure bölge/SKU uygunluk kontrolü, kota kontrolü

**Örnek:** "Projenin mimari diyagramını oluştur", "Containerization planı hazırla"

---

## 5. Pylance MCP (`mcp_pylance_mcp_s_*`)

Python geliştirme ortamı için Pylance dil sunucusu entegrasyonu sağlar.

**Temel Kullanım Alanları:**
- Docstring üretimi, import yönetimi, syntax hata kontrolü
- Python environment yönetimi, refactoring, kod çalıştırma

**Not:** Bu proje Java/Spring tabanlı olduğundan bu MCP seti doğrudan kullanılmaz.
