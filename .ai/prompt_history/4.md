## Root Cause Analizi

**Sorun:** `springdoc.api-docs.enabled: false` ayarı, springdoc'un Swagger UI altyapısını da pasifleştiriyor. Springdoc 2.x'te `SwaggerWelcomeWebMvc` controller'ı ve `/swagger-ui/**` resource handler'ı (`SwaggerWebMvcConfigurer`) api-docs bean zincirine bağımlı; bu yüzden Swagger UI webjars hiç serve edilmiyor.

**Hata mesajlarının açıklaması:**
- `No static resource swagger-ui.html` → springdoc'un redirect controller'ı kaydedilmedi
- `No static resource swagger-ui/index.html` → springdoc'un webjar resource handler'ı kaydedilmedi
- `No static resource sw.js` → yukarıdakinin cascade etkisi

---

**Tek değişiklik:** application.yml içinden `api-docs.enabled: false` kaldırılacak.
